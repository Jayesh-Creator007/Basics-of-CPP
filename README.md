# Basic of C++ Language

This repository demonstrates basic concepts of the C++ programming language including output, variables, data types, and boolean values. The goal is to help beginners understand how data is stored and displayed in C++.

## Concepts Covered

- cout output statements
- Integer, Float, Double, Char data types
- Boolean values
- string usage in C++
- Basic formatting using endl
- Difference between C and C++ output syntax

## Requirements

Any C++ compiler such as:
- GCC / G++
- MinGW
- Clang++
- Visual Studio
- Code::Blocks
- Dev-C++
